Vortrag: Effektives Monitoring von Linux und Services

In zunehmend komplex werdenden Infrastrukturen wird effektives Monitoring immer wichtiger. Aber auch für kleine Setups ist Monitoring bereits essentiell. Dabei müssen Punkte wie spezifische Grenzwerte, Performance sowie Benachrichtigungen beachtet werden.

Zielgruppe: Interessierte technisch versierte Personen, Linux Anwender
Vorkenntnisse: Man sollte wissen wie Computer und das Internet
funktionieren. Idealerweise hat man Linux Grundkenntnisse.
Thema des Talks:
- Für was man Monitoring braucht
- Warum das jeder braucht, der einen Server/Service betreibt
- Was soll man überwachen?
- Warum effektives Monitoring keine einfache Aufgabe ist
- Zu was führt falsch konfiguriertes Monitoring
- Wie man effektives Monitoring konfiguriert (Am Beispiel Nagios/Icinga)
- Ausblick auf komplexe Infrastrukturen
